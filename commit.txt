
1. do_cmake.sh modified 
Change the installation path to replace executables installed by apt-get. 
To avoid tricky configurations in execution, we create a private apt-get repository 
and install ceph from it. Then, we replace the modified version of Ceph with it. 

<< 
cmake -DBOOST_J=$(nproc) "$@" ..

>>
cmake -DCMAKE_INSTALL_PREFIX:PATH=/usr -DBOOST_J=$(nproc) "$@" ..
